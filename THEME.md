# 🎨 Unified Research Identity Theme

This document defines the visual identity for the **Juan Moisés de la Serna** research ecosystem. By applying this theme, all repositories (main portal, library wikis, etc.) will have a consistent, professional, and academic presence.

## 🌈 Core Palette
- **Primary (Academic Blue)**: `#0366d6`
- **Secondary (Dark Blue)**: `#0056b3`
- **Background (Main)**: `#ffffff`
- **Background (Alt/Cards)**: `#f6f8fa`
- **Accent (Tags/Tags)**: `#e1ecf4` / Text: `#39739d`
- **Borders**: `#e1e4e8`

## 🖋️ Typography
- **Headings & Body**: `Inter` (sans-serif)
- **Code & Citations**: `JetBrains Mono` (monospace)

## 🏗️ Visual Components
- **Header**: Large title (`2.5rem`), strong weight (`700`), blue color.
- **Section Dividers**: Bottom border (`2px solid #e1e4e8`) with padding.
- **Cards**: Background `#f6f8fa`, border `1px solid #e1e4e8`, corner radius `12px`, soft shadow.
- **Topic Tags**: Rounded pills (`20px`), background `#e1ecf4`, text `#39739d`.
- **Code/BibTeX Blocks**: Dark theme background `#24292e`, text `#e6edf3`, rounded corners.

## 🛠️ Implementation
To apply this theme, include the global CSS (located in `assets/css/research-identity.css`) and follow the HTML structure defined in `index.html`.
