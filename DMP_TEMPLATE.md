## 2. FAIR Principles Compliance
This project strictly adheres to the **FAIR Data Principles**:
- **Findable:** Datasets are hosted on GitHub and indexed in Mendeley Data/Zenodo with unique DOIs.
- **Accessible:** All data is publicly available via standard HTTP protocols.
- **Interoperable:** Data uses open formats (CSV, JSON) and standardized metadata (Schema.org, Dublin Core).
- **Reusable:** Data is licensed under **CC0 1.0 Universal**, allowing for unrestricted reuse and redistribution.
