# Recomendaciones de Imagen Destacada (Pilot 15)

Para mantener la identidad visual del Dr. Juan Moisés de la Serna, se recomiendan imágenes de estilo **ilustrativo, conceptual y minimalista**, evitando fotografías literales de stock. El uso de esquemas neuroanatómicos estilizados o metáforas visuales limpias es preferible.

| Post | Recomendación de Imagen |
| :--- | :--- |
| **Sistema glinfático** | Ilustración 3D del cerebro con un flujo de líquido azul luminoso atravesando los tejidos (efecto "lavado"). |
| **Horas de dormir** | Un reloj de arena donde la arena se transforma en una red neuronal en la parte inferior. |
| **Luz azul pantallas** | Perfil humano frente a un dispositivo que emite ondas azules que impactan directamente en una estructura cerebral brillante (NSQ). |
| **TDAH mujeres adultas** | Un caleidoscopio con forma de cerebro, simbolizando la complejidad y el enmascaramiento. |
| **TDAH adultos: síntomas** | Un tablero de mandos (director de orquesta) con algunos cables desconectados o luces parpadeantes. |
| **TDAH: medicación mito** | Una llave de precisión química encajando en una cerradura con forma de receptor dopaminérgico. |
| **Neurobiología memoria** | Una red de nodos luminosos donde algunos puntos se encienden y otros se apagan suavemente. |
| **Memoria de trabajo** | Una pizarra mental con 4-5 elementos flotando en el centro de una silueta humana. |
| **Mitos de la memoria** | Un puzle de un paisaje donde algunas piezas están siendo reubicadas o tienen colores diferentes. |
| **Adicción digital** | Una mano sosteniendo un smartphone del que salen hilos invisibles (como una marioneta) hacia el cerebro. |
| **Multitarea: mito** | Un malabarista intentando manejar demasiadas esferas luminosas, con algunas empezando a caer. |
| **IA y cerebro** | Dos redes neuronales (una biológica y una artificial) entrelazándose sin perder su identidad. |
| **Estrés y eje HPA** | Un esquema estilizado del eje (cerebro-riñones) con una señal de alerta pulsante. |
| **Neurobiología resiliencia** | Una planta creciendo a través de una red neuronal geométrica y rígida. |
| **Eustrés vs Distrés** | Una cuerda de violín tensa en el punto justo para emitir una nota musical (onda armónica). |
