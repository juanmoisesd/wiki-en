## 1. Data Summary
- **Purpose:** To provide a centralized Research Wiki and Open Dataset Collection for forensic neuroeconomics and AI research.
- **Data Type:** Markdown documentation, JSON-LD metadata, and CSV dataset references.
- **Expected Users:** Academic researchers, data scientists, and policy makers interested in economic decision biases and anti-fraud systems.

## 2. FAIR Principles Compliance
This project strictly adheres to the **FAIR Data Principles**:
- **Findable:** Datasets are hosted on GitHub and indexed in Mendeley Data/Zenodo with unique DOIs.
- **Accessible:** All documentation and terminology are publicly available via GitHub Pages.
- **Interoperable:** Uses standardized metadata (Schema.org, Dublin Core, Highwire Press).
- **Reusable:** All content is licensed under **CC0 1.0 Universal**, allowing for unrestricted reuse.

## 3. Storage and Preservation
- **Primary Storage:** GitHub (Public Repository).
- **Redundancy:** Synchronized with Zenodo (DOI: 10.5281/zenodo.19415092).
